# 应用场景（现场演示）

开发与非开发场景的典型用法，从工程到知识工作。

- 专利写作代理（Patent Writing Agent）：从技术说明到权利要求书草拟与结构化输出。
  - 示例（私有仓库）：https://codeup.aliyun.com/61adcb80e05da4a409ab67b8/intellif/job-daylog
- 旅行代理（Travel Agent）：多城市行程规划与订票辅助，自动整合交通与住宿信息。
  - 示例（私有仓库）：https://github.com/igamenovoer/travel-plan-202510
- 论文综述代理（Paper Survey Agent）：读取论文与资料，生成主题综述与参考清单。
  - 示例：https://github.com/imsight-knowhow/yolo13-reading
