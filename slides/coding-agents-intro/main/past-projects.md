# 过往项目回顾

回顾使用编码代理完成的实践项目。

- LLM 性能测试（llm-perf-opt）：测试LLM的内核性能，给NPU设计提出建议
  - GitHub：https://github.com/CodeGandee/llm-perf-opt#
- InvokeAI Python 客户端（invokeai-py-client）：通过 Python 调用/编排 InvokeAI 进行图像生成
  - GitHub：https://github.com/CodeGandee/invokeai-py-client
  - PyPI：https://pypi.org/project/invokeai-py-client/
- PeiDocker（pei-docker）：便于定制和复用的 Docker 镜像构建工具
  - GitHub：https://github.com/igamenovoer/PeiDocker
  - PyPI：https://pypi.org/project/pei-docker/
- Blender Remote（blender-remote）：通过 MCP 远程控制 Blender，支持 AI 助手编排建模/渲染流程
  - GitHub：https://github.com/igamenovoer/blender-remote
  - PyPI：https://pypi.org/project/blender-remote/
