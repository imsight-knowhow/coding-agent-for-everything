#### 1. "AI Agent 元年"之争
- **媒体宣称**：2025 是"AI Agent 元年"
- **现实情况**：Gartner 将 AI Agent 标注在"期望膨胀峰值"阶段
  <br/>[[Gartner - Hype Cycle 2025]](https://www.gartner.com/en/newsroom/press-releases/2025-08-05-gartner-hype-cycle-identifies-top-ai-innovations-in-2025)
- **专家观点**：IBM 专家质疑"连 LLM 的 ROI 都没算清，就谈 Agent？"
  <br/>[[IBM - AI Agents: Expectations vs. Reality]](https://www.ibm.com/think/insights/ai-agents-2025-expectations-vs-reality)

</div>

#### 2. 没有通吃的工具：Cursor vs Claude Code vs Cline
- **IDE 派（Cursor）**：图形界面友好，部署能力强
  <br/>[[Qodo - Deep Comparison]](https://www.qodo.ai/blog/claude-code-vs-cursor/)
- **CLI 派（Claude Code）**：终端优先，200k token 上下文
  <br/>[[HaiHai AI - Cursor vs Claude Code]](https://www.haihai.ai/cursor-vs-claude-code/)
- **成本争议**：相同模型下 Claude Code 比 Cursor 贵
  <br/>[[AIMultiple - Agentic CLI Tools]](https://research.aimultiple.com/agentic-cli/)
- **性能相近**：两者基于 Claude 3.7 Sonnet，质量相似
  <br/>[[Render - AI Coding Agents Benchmark]](https://render.com/blog/ai-coding-agents-benchmark)

</div>
<div>

#### 3. 生产力数据 vs 炒作担忧
- **✅ 实测**：4,867 名开发者，任务量 +26%，代码更新 +13.55%
  <br/>[[Index.dev - AI Agent Statistics]](https://www.index.dev/blog/ai-agents-statistics)
- **⚠️ 狂热**：2025 年市场 73.8 亿美元，2023 年的两倍
  <br/>[[Pragmatic Coders - 200+ Statistics]](https://www.pragmaticcoders.com/resources/ai-agent-statistics)
- **❌ 顾虑**：87% 担忧准确性，81% 担忧安全
  <br/>[[Index.dev - Adoption Trends]](https://www.index.dev/blog/ai-agents-statistics)

#### 4. "自主性"神话
- **宣传**：完全自主的 AI 编码代理即将到来
- **现实**：多数"自主 AI"演示离开人工帮助会失败
  <br/>[[Thoughtworks - Macro Trends]](https://www.thoughtworks.com/insights/blog/technology-strategy/macro-trends-tech-industry-november-2025)
- **真相**：仍处实验阶段，可靠性与真正自主性受限
  <br/>[[Alvarez & Marsal - Separating Hype from Reality]](https://www.alvarezandmarsal.com/thought-leadership/demystifying-ai-agents-in-2025-separating-hype-from-reality-and-navigating-market-outlook)

## 调研机构：肯定有用，但要小心用

- **采用率高**：85% 组织已将 AI Agent 集成到至少一个工作流
  <br/>[[Index.dev - Statistics]](https://www.index.dev/blog/ai-agents-statistics)
- **效果明显**：26% 生产力提升有真实数据支持
  <br/>[[Pragmatic Coders - 200+ Statistics]](https://www.pragmaticcoders.com/resources/ai-agent-statistics)
- **仍在探索**：可靠性、泛化能力、长期自主性仍受限
  <br/>[[Alvarez & Marsal - Market Outlook]](https://www.alvarezandmarsal.com/thought-leadership/demystifying-ai-agents-in-2025-separating-hype-from-reality-and-navigating-market-outlook)
- **工具选择**：没有绝对最佳，取决于工作流偏好（IDE vs CLI）
  <br/>[[Artificial Analysis - Comparison]](https://artificialanalysis.ai/insights/coding-agents-comparison)
