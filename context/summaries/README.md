# Summaries 目录

## 用途

维护分析文档、项目知识总结以及研究或实施工作的综合发现。

## 内容类型

- 技术选型分析
- 架构决策理由
- 实施经验教训
- 性能分析报告
- 安全审计总结

## 命名规范

使用描述性分析主题：
- `library-comparison-analysis.md`
- `architecture-decision-rationale.md`
- `implementation-lessons-learned.md`
- `performance-benchmark-summary.md`

## 示例

```markdown
# 库对比分析

## HEADER
- **Purpose**: 比较三个候选库以选择最佳方案
- **Status**: Completed
- **Date**: 2025-01-15
- **Dependencies**: 库文档、性能测试结果
- **Target**: 技术决策参考
```
