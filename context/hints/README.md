# Hints 目录

## 用途

创建常见开发任务、错误解决方案和项目特定最佳实践的"操作指南"。帮助 AI 助手避免已知陷阱。

## 内容类型

- 环境设置指南
- 常见错误排查
- 调试技巧
- 性能优化建议
- 最佳实践说明

## 命名规范

使用面向操作的前缀：
- `howto-setup-development-environment.md`
- `howto-debug-connection-issues.md`
- `why-this-error-occurs.md`
- `troubleshoot-build-failures.md`

## 示例

```markdown
# 如何设置开发环境

## HEADER
- **Purpose**: 指导新开发者设置本地开发环境
- **Status**: Active
- **Date**: 2025-01-15
- **Dependencies**: Docker, Node.js
- **Target**: 新团队成员和 AI 助手
```
