# Instructions 目录

## 用途

存储可复用的提示词片段、命令模板和标准化指令模式，可在整个项目中引用。有助于保持 AI 交互的一致性。

## 内容类型

- 代码审查模板
- Git 工作流命令
- 错误处理模式片段
- 功能文档模板
- 标准化提示词

## 命名规范

使用命令或片段类型前缀：
- `prompt-code-review-template.md`
- `command-git-workflow.md`
- `snippet-error-handling-pattern.md`
- `template-feature-documentation.md`

## 示例

```markdown
# 代码审查提示词模板

## HEADER
- **Purpose**: 提供标准化的代码审查提示词
- **Status**: Active
- **Date**: 2025-01-15
- **Dependencies**: 无
- **Target**: AI 助手进行代码审查
```
