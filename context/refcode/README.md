# RefCode 目录

## 用途

存储参考实现、代码示例和第三方集成，作为新开发的模式参考。

## 内容类型

- 参考实现代码
- 第三方 API 集成示例
- 设计模式实现
- 最佳实践代码片段
- 外部库使用示例

## 文件组织

可以按技术栈、功能或模式组织：
- `authentication/jwt-example.js`
- `database/pagination-pattern.sql`
- `api/rest-client-example.py`
- `patterns/observer-implementation.ts`
